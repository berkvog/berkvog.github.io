Whiskr

Whiskr was an app myself and other classmates built in school. Whiskr is a mobile app that allows users to enter the contents of their pantry or fridge, and then see what recipes they can make!

I personally worked on backend database setup as well as html parsing for our site gather recipes. This app was a ton of fun to build!

Context Engine

Worked for the SEE-lab at the University of California at San Diego developing on machine learning algorithms. The Context Engine is a complex data analysis framwork that essentially allows for multi-input, multi-output data analysis on embedded systems. It contains a suite of algorithms tailored to SmartGrid analysis.

I personally implemented the Agile Anomaly Detection algorithm which was originally transposed from the twitter S-H-ESD algorithm in R.
